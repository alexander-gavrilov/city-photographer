# Story: Visual Photo Feedback

**Feature:** [Enhanced Camera Feedback](../f007_enhanced_camera_feedback.md)

**Description:** Implement visual effects that provide feedback when photos are taken, especially when clues are successfully captured.

**Requirements:** Create visual effects that enhance the photo-taking experience and clearly indicate successful captures.

**Constraints:** Visual effects should be noticeable but not overwhelming or distracting from gameplay.

**Acceptance Criteria:**
*   [ ] Camera flash effect when photos are taken
*   [ ] Visual distinction between successful vs unsuccessful photos
*   [ ] Special visual effect for capturing rare or sequence clues
*   [ ] Camera viewfinder/frame overlay during photo mode
