# Task: Include meta tags

**Story:** [HTML Page Setup](../s001_html_page.md)

**Description:** Add meta tags for character encoding and viewport settings.

**Requirements:** Include charset and viewport meta tags.

**Status:** ✓ DONE

**Acceptance Criteria:**
- [x] UTF-8 charset meta tag is present in the head
- [x] Viewport meta tag is present with width and initial-scale settings

**Implementation Details:**
```html
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

**Location:** These tags should be placed in the `<head>` section of index.html
