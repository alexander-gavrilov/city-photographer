# Story: Sequential Puzzles

**Feature:** [Advanced Clue System](../f006_advanced_clue_system.md)

**Description:** Implement puzzles that require photographing multiple clues in a specific sequence to trigger reactions or unlock areas.

**Requirements:** Create a system to track the order of photographed clues and validate sequences.

**Constraints:** The sequence requirements should be discoverable through environmental hints or visual cues.

**Acceptance Criteria:**
*   [ ] System for tracking sequence of photographed clues
*   [ ] At least 2 sequential puzzles implemented in different districts
*   [ ] Environmental hints that suggest the correct sequence
*   [ ] Feedback when sequence steps are completed correctly/incorrectly
