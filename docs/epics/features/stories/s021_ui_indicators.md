# Story: UI Indicators

**Feature:** [Enhanced Camera Feedback](../f007_enhanced_camera_feedback.md)

**Description:** Implement UI elements that provide information about camera targeting, successful captures, and clue tracking.

**Requirements:** Create minimal but effective UI indicators that inform players without breaking immersion.

**Constraints:** UI should be subtle and fit the game's aesthetic, avoiding cluttered or obtrusive elements.

**Acceptance Criteria:**
*   [ ] Targeting indicator showing when a clue is in focus
*   [ ] Capture confirmation UI element
*   [ ] Collection/progress tracking for clues
*   [ ] Hint system for sequential puzzles
